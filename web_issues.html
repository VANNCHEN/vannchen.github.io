MY perception of web page improvement


  • Proper picture

    • A web page should have more than just plain text. It should also have appropriate images. And to match the overall tone and to match the background of the page. And the image should correspond to the theme, the image size should be reasonable.

  • The navigation bar of the sub-page

    • Navigation bars should not be limited to the main page, but should be present on every page. This will be very convenient for visitors to visit. And the best navigation bar with the home page navigation bar style and position consistent, so as to maintain the visual beauty.

  • Use lists or tables

    • A good web page can not only rely on words to show, should be free to use lists or tables to show the content. A complete form can clearly express what you want to show, making it easier for visitors to find useful information.
